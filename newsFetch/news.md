# AI Developer News: June 9–11, 2026

## Model Releases
*   **DiffusionGemma (Google):** An experimental open model (Apache 2.0) that utilizes text diffusion to achieve up to 4x faster inference than standard LLMs on dedicated GPUs. It is a 26B parameter Mixture of Experts (MoE) model.
*   **Gemini 3.5 Live Translate (Google):** A new audio model for near real-time speech-to-speech translation in over 70 languages, available to developers via the Gemini Live API and Google AI Studio.
*   **North Mini Code (Cohere):** An open-source, 30B parameter MoE coding agent designed for agentic software engineering. It runs on a single H100 and is optimized for terminal-based tasks, architecture mapping, and code review.
*   **Claude Mythos 5 & Fable 5 (Anthropic):** Anthropic released Mythos 5, a restricted-access model with advanced cybersecurity capabilities, alongside Fable 5, a safer version designed for general public use.

## Tech & Feature Releases
*   **Claude Managed Agents (Anthropic):** Added support for scheduled deployments (cron jobs) and secure credential vaults (environment variables), allowing agents to run autonomously on a schedule without external infrastructure.

## Acquisitions & Investments
*   **NEURA Robotics:** Secured $1.4 billion in funding, backed by Tether, Nvidia, and Amazon, for their work in humanoid robotics.
*   **Kumo AI:** Acquired by NVIDIA for a reported $400 million. Kumo specializes in foundation models for relational data and enterprise applications.
*   **Standard Bots:** Raised $200 million in a Series C round at a $1 billion valuation. The company specializes in AI-native robotic arms for industrial applications.
*   **Beacon:** Raised $225 million in a Series C round to further its "AI-native" software holding company model, which modernizes legacy vertical software using AI.
*   **Anthropic Infrastructure Deal:** Google has agreed to backstop lease payments for a $35 billion chip financing deal, securing Anthropic's long-term access to Google's TPU ecosystem.
